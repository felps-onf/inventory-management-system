# Inventory Management System

A simple console-based inventory management application written in C#.

## Features

- View all products in the inventory
- Add new products
- Update stock quantities
- Remove products
- Run built-in validation tests

## Requirements

- .NET SDK 10.0 or later

## How to run

1. Clone the repository
2. Navigate to the project folder
3. Run:

```bash
dotnet run
```

## Build

To build the project:

```bash
dotnet build
```

## Notes

The application includes a built-in self-test mode that validates the core inventory operations.
